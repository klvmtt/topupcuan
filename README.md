# TOPUPCUAN Telegram Landing Page

Landing page Telegram bergaya neon emas/kuning untuk GitHub Pages.

## Cara upload ke GitHub Pages

1. Upload `index.html`, `style.css`, dan `script.js` ke repository GitHub.
2. Masuk ke **Settings > Pages**.
3. Pilih branch `main` dan folder `/root`.
4. Save, lalu buka URL GitHub Pages yang diberikan.
